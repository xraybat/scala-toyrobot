# scala-toy-robot

scala toy robot

## to run in general

from OS CLI (either):

    $ sbt "run <filename>"
    $ sbt run

from within  sbt:

    sbt> run <filename>
    sbt> run

and similar for testing:

    $ sbt test
    sbt> test

## to see the three examples in PROBLEM.md

    $sbt "run src/resources/eg1.txt"
    $sbt "run src/resources/eg2.txt"
    $sbt "run src/resources/eg3.txt"
